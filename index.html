SEMÁFORO de riesgo epidemiológico

Te mostraremos como está la situación ante la pandemia que estamos viviendo, para esto decidimos mostrar un semáforo en el cual dependiendo del color es la gravedad con la que estamos combatiendo.

ROJO

Solo se  tiene permitido únicamente las actividades económicas esenciales.

NARANJA

Solo laboran actividades económicas esenciales y no esenciales, pero ésta última solo con el 30% del personal para su funcionamiento

AMARILLO

Con color amarillo esta permitido realizar las actividades laborales, y se deben de realizar con las medidas básicas de prevención.

VERDE

Este color nos permite realizar todas nuestras actividades, tanto para adultos como niños.
